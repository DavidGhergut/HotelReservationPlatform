# Hotel Reservation Platform

## Used tech stack

- Frontend:
  - React
  - Typescript
  - Vite as build tool
 
- Backend
  - Java
  - Spring Boot (+ Lombok)

- Database: H2 database

## Description
This is my attempt at creating a hotel reservation page where the user can see his location and filter out hotels in the nearest vicinity or further.
